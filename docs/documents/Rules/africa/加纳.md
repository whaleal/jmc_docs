__SenderID规则__

| SenderID类型     | 规则                                                                          |
|----------------|-----------------------------------------------------------------------------|
| 字母数字混合SenderID | 支持字母数字发件人 ID，MTN (62001) 除外。*MTN (62001) 网络必须进行预注册。发件人 ID 区分大小写，并且必须包含品牌名称。 |
| 数字SenderID     | 	除非 Vonage 的 2-way 服务提供，否则不支持数字发件人 ID 。数字发件人 ID 将被覆盖以确保送达。                  |
| 通用SenderID     | 禁止使用通用发件人 ID，例如 INFO、SMS、NOTICE 等。                                          |
