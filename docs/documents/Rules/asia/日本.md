__带有 URL 链接的消息内容__

日本网络针对网络钓鱼短信采取了预防措施，这些措施可能会影响带有 URL 链接的消息的传递。日本网络不会透露他们的拒绝标准。

__连接 Unicode 消息__

对于 Docomo、KDDI 和 Softbank 网络，unicode 消息的最大长度/字符为 670 个全角字符，对于 Rakuten 网络为 660 个全角字符。
