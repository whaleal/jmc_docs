__SenderID规则__

| SenderID类型     | 规则                                     |
|----------------|----------------------------------------|
| 字母数字混合SenderID | 由于垃圾邮件拦截，所有字母数字发件人 ID 都必须预先注册*。        |
| 数字SenderID     | 除非 Vonage 的 2-way 服务提供，否则不支持数字发件人 ID 。 |
| 通用SenderID     | 	禁止使用通用发件人 ID，例如 INFO、SMS、NOTICE 等。    |
