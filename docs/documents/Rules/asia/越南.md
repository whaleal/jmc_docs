__内容过滤__


短信内容可能会被修改，例如删除品牌名称或修改一些特殊字符，例如“代码”，或者在消息正文前面添加随机字符等，以绕过过滤并提高发送率。对于带有 OTP 代码的消息，最终用户也可能会收到传递 OTP 代码的文本转语音语音呼叫，而不是接收 SMS 消息。

强烈建议注册您的发件人 ID 和内容，以避免任何更改或递送问题。
