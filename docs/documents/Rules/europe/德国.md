__SenderID规则__

| SenderID类型     | 规则                                                                 |
|----------------|--------------------------------------------------------------------|
| 字母数字混合SenderID | 完全支持 Alpha 发件人 ID。                                                 |
| 数字SenderID     | 支持数字发送者 ID，LVN 也可以由Vonage 的 2 向服务 提供 。短代码号码不能在 Vodafone 26202 上使用。 |
| 通用SenderID     | 禁止使用通用发件人 ID，例如 INFO、SMS、NOTICE 等。请尽量使用有代表性的品牌名称，以确保更好的交付。         |
